Basic Usage
```jsx
<Button label='Submit'/>
```

Disabled
```jsx
<Button label='Submit' disabled/>
```

Theme
```jsx
<div>
    <Button label='primary' theme="primary"/>
    <Button label='secondary' theme="secondary"/>
    <Button label='danger' theme="danger"/>
</div>
```

Inverse
```jsx
<Button label='Submit' inverse theme="primary"/>
```
