### Example

You can add as many tabs as you would like, the bottom sliding line will adjust accordingly. Must contain 1 tab.

#### Basic Usage
```jsx
const AccordionItem = require('./index').Tab;

<Tabs startIndex={0}>
    <Tab label="Home"/>
    <Tab label="About"/>
    <Tab label="FAQ"/>
    <Tab label="Careers"/>
    <Tab label="News"/>
</Tabs>
```
